# Makers' Software Sector Repository

This repository is designed for our school's CCA Software Sector to collaborate, learn, and grow together.

## Purpose

The goal of this repository is to:
- Provide **reference solutions** from seniors  
- Allow juniors to **practice and submit their own solutions**  
- Encourage **learning through comparison and improvement**  
- Build a shared **knowledge base** over time  

## How It Works

### For Seniors
- Add well-structured and clean solutions to problems
- Include comments and explanations where necessary
- Organize files clearly

### For Juniors
- Attempt problems on your own first  
- Upload your solutions to the repository  
- Compare your work with senior solutions  
- Learn and improve from feedback

## Getting Started - For Juniors

1. Fork this repository  
2. Clone your fork:
   
       git clone https://github.com/your-username/repo-name.git

3. Add your solution files 

        git add your-name/

4. Commit and push:
   
       git commit -m "Added solution for [problem name]"
       git push

5. Open a Pull Request

## Guidelines

- Write **clean and readable code**
- Add **comments** where needed
- Use meaningful **file and folder names**
- Do not copy solutions without understanding them
- Be respectful when reviewing others' work

## Contributions

All members of the Software Sector are encouraged to contribute!  
Seniors can guide by example, and juniors can improve through practice.

## Notes

- This is a **learning-focused repository**, not just for correct answers  
- Mistakes are welcome — they’re part of the process
